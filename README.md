# 🧮 Digital Calculator

A simple and stylish digital calculator built using HTML, CSS, and JavaScript.

---

## 🚀 Features

* Basic arithmetic operations ➕ ➖ ✖ ➗
* Responsive design 📱💻
* Clean and modern UI 🎨
* Error handling for invalid inputs ⚠️
* Decimal support 🔢

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
